核心功能

保留原版 + 自部署增强

与 Readest 功能持续对齐，所有原版 Pro 专属功能无条件开放；额外内置多用户、远程下载、Web 搜索、翻译代理、Edge TTS 等自部署增强能力。

核心阅读能力

📚

多格式阅读

原生支持 EPUB、PDF、MOBI、AZW3 等主流电子书格式，无需转换直接打开，自动适配最佳排版。

🖥

分屏阅读

原版 Pro 功能，现无条件开放。支持双屏对照阅读，左原文右译文，或同时参考两本书。

🎤

TTS 朗读

原版 Pro 功能，现无条件开放。内置高质量语音合成，多语种多音色，可调语速音调。

☁

云端同步

阅读进度、批注、书签自动同步，多端无缝切换。同步协议完全兼容原版前端逻辑。

✍

无限批注与导出

原版 Pro 功能，现无条件开放。高亮、划线、笔记一应俱全，支持导出 Markdown 或纯文本。

🗃

实际无上限存储

100TB 软上限（不强制），实际只受磁盘空间约束。书籍、封面存于本地挂载卷，数据完全自主。

自部署增强 v7.0 · v8.0 · v8.1 · v8.2 · v8.3 · v8.4 · v8.5 · v8.6 新增

👥

多用户管理

管理员可创建普通用户，分配存储与翻译配额。v8.4：Per-user 加密 vault，library/settings 按 userId AES-GCM 加密，密码不存客户端，跨账号彻底隔离。v8.5：配额真正 enforce，超额上传/翻译被拒绝。普通用户独立空间，互不干扰。

📥

远程下载

书库点「导入」→「Download from URL」，服务器自动抓取入库。v8.6：任务队列，有进度/状态/重试，下载完成后多端同步可见。

🌐

翻译词典代理

Google 翻译、Wikipedia、Wiktionary 全部走服务器代理，客户端无需加速器。v8.0：强制登录。v8.2：可关代理。v8.5：SSRF 黑名单。

📚

原版词典悬浮窗

选中文字点「Dictionary」弹悬浮窗显示 Wikipedia/Wiktionary 查询结果，点「Translate」弹翻译悬浮窗。保留原版 Readest 完整体验，代理透传。

🔊

Edge TTS

服务器端 WebSocket 连接微软 Edge TTS，高质量语音合成，多语种多音色，零成本零依赖。

🔤

字体本地化

阅读字体打包进镜像，不依赖外部 CDN，离线环境也能保持一致的中文渲染效果。

改动明细

只做减法，不加戏

严格遵循最小化改动原则：删除付费体系与注册限制，其余逻辑原封不动保留。

✕

已移除

  • −Pro 付费体系、定价页、订阅页、支付页面
  • −所有升级弹窗、Pro 标识徽章、升级入口
  • −isPro / checkPro / requirePro 校验代码
  • −注册功能入口，仅管理员可创建用户
  • −存储容量限制、文件数量限制
  • −Supabase Auth 依赖，替换为本地鉴权
  • −Supabase Postgres 依赖，替换为 SQLite
  • −对象存储依赖，替换为本地文件系统
✓

保留 / 新增

  • +全部阅读器内核、UI 交互、同步逻辑原封不动
  • +原 Pro 功能（分屏、TTS、导出批注）全部开放
  • +无限存储，无容量与文件数限制
  • +多用户系统，管理员可创建用户并分配配额
  • +远程书籍下载，服务器代理获取 URL 书籍（v8.1：同时写 Book 表，多端同步可见）
  • +翻译词典服务器代理，客户端无需加速器（v8.0：强制登录）
  • +Edge TTS 服务器端合成，零成本零依赖
  • +字体本地化，不依赖外部 CDN
  • +DeepL 翻译可选启用（默认关闭）
  • +100TB 软上限（实际只受磁盘约束）
  • +SQLite + Prisma 替代 Postgres，数据完全自主
  • +本地文件系统替代对象存储，书籍存于 /data/books
  • +全部 API 契约 1:1 复刻，前端零修改可对接
  • +单 Docker 镜像，端口 8225，数据卷持久化
